📱 Bem-vindo à tua aplicação Expo 👋

Este é um projeto criado com Expo através do comando create-expo-app.

O Expo facilita o desenvolvimento de aplicações React Native que funcionam em Android, iOS e Web.

🚀 Como começar
1️⃣ Instalar dependências

No terminal, dentro da pasta do projeto, executa:

npm install

2️⃣ Iniciar a aplicação
npx expo start

Após o arranque, tens várias opções para testar a app:

📲 Expo Go → abre no teu telemóvel Android/iOS através do QR code

🤖 Emulador Android → usar via Android Studio

🍏 Simulador iOS → disponível apenas em macOS

🛠️ Development Build → para testar funcionalidades mais avançadas

📂 Estrutura do projeto

O código principal está dentro da pasta /app.

Este projeto usa file-based routing (as rotas são criadas automaticamente com base nos ficheiros).

Para começar a desenvolver, edita os ficheiros dentro da pasta app.

🧹 Resetar o projeto

Se quiseres começar do zero:

npm run reset-project

Isto vai mover o código de exemplo para a pasta /app-example e criar uma nova pasta /app vazia.
